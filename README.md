Aviation × Data Strategy Portfolio

A highly customized, high-performance personal portfolio website built to showcase the strategic intersection of **Aviation Management** and **Management Information Systems (MIS)**. Designed with a bespoke, airport-instrumentation-inspired dark UI theme, this project serves as a comprehensive digital resume and vision statement for high-level business intelligence, data strategy, and future holding management roles.

## 🛩️ Project Philosophy & Design Language

Unlike generic, boilerplate templates, this portfolio is built completely from scratch using a precise aesthetic framework:
- **The Instrument-Strip Grid:** Structural layouts inspired by cockpit control panels and modern data terminals.
- **The Split-Flap Display:** A smooth, custom JavaScript role-rotator mimicking classic airport departures/arrivals boards.
- **Advanced Typography Contrast:** Leveraging *Fraunces* (for editorial elegance), *Inter* (for high UI legibility), and *IBM Plex Mono* (for raw data/technical precision).
- **Control Tower Palette:** Utilizing a deeply refined dark foundation (`#0a0e16`) accented with functional, high-visibility tactical amber (`#f2b84b`) and deep slate accents.

## 🛠️ Core Features & Components

### 1. Split-Flap Role Rotation (`js-split-flap`)
A custom-built, vanilla JavaScript utility that dynamically transitions through professional titles (**Data Analyst**, **Business Analyst**, **Future Tech Strategist**). It uses precise `setTimeout` synchronization and `requestAnimationFrame` to prevent layout shift and guarantee buttery-smooth performance.
- Fully accessible with `prefers-reduced-motion` responsive fallback detection.

### 2. Analytical Skill Matrix (`data-pct` Data Binding)
A visual metrics grid displaying core proficiencies (SQL, Python, Power BI, Tableau, Advanced Analytics). 
- **Under the Hood:** Leverages a custom HTML5 dataset configuration combined with an asynchronous `IntersectionObserver` in JavaScript. Progress bars animate gracefully from `0` to their exact target percentages *only when* they enter the viewport.

### 3. Responsive Operational Layout
Built with defensive modern CSS (`box-sizing: border-box`, clean flex/grid fluid dynamics) making it completely optimized across all viewports—from 4K desktop command monitors down to mobile screens.

## 🧮 Tech Stack & Engineering Specs

- **Structure:** Semantic HTML5 (Accessible layout wrappers, metadata tag engine)
- **Styling:** Vanilla CSS3 (Custom properties / design tokens, performance-optimized transforms, hardware-accelerated animations)
- **Interactivity:** Vanilla JavaScript (ES6+ Asynchronous API observers, DOM manipulation engines)
- **Design System:** Custom typography tokens, explicit line-height bounding (1.15 - 1.25 for absolute readability), and high-contrast color spaces.
